This is a Contact Management App built with React 

How to Run the Project
1) clone the repo :git clone https://github.com/andrewref/contact-management-app.git

2) Navigate to the project folder: cd contact-management-app

3) Install dependencies: npm install

4) Run: npm run dev

Design Decisions

contacts are intialized using a mock JS file 
all add,edit,and delete operations are handled in memory
Component-Based Architecture
The app is split into small, reusable components such as:

ContactList
ContactItem
ContactForm
CategoryFilter

Also made the same form for edit and adding contacts
